Page Fault гэж юу вэ

  • Page RAM-д байхгүй → Storage-оос татаж ачаална
  • Soft page fault – page RAM-д байгаа ч өөр процессийн хуваарилалттай
  • Hard page fault – page RAM-д байхгүй, дискнээс ачаалах шаардлагатай
  • Page fault cycle – fault илрэх, MMU шалгах, дискнээс ачаах, RAM-д байрлуулах
